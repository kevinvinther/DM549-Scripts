# Truth	table Generator
generere et truth table med logiske expression :)

download python 3, hvis det ikke allerede er på maskinen :D
derfter kan du køre `Truth table generator.py`

# logiske udtryk
- `!`, `~`, `not`, `¬` er the samme.
- `&`, `*`, `and`, `∧`, er det samme.
- `|`, `+`, `or`, `∨`, `v` er det samme.
- `<->`, `↔` er det samme.
- `->`, `→` er det samme.
- `^`, `⊕` er det samme.
